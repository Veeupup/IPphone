# IP电话

暂时的思路：

使用P2P的架构，每个客户端进行两个进程，分别是

* 作为服务端，
  * 在线，等待其他的用户拨号，并且可以选择是否接受电话
  * 当正在通话的时候，拒绝其他的连接
* 作为客户端
  * 去连接其他的用户，如果返回的状态是接受，那么二者进入通话状态，这个时候开始传送语音
